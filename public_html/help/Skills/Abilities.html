Help Topic: Abilities

SYNTAX:

abilities <category>            : View skills you know in a category.
abilities -l <category> <class> : View all skills for a class in a category,
                                  sorted by level.

Categories:         Classes:

Language            Bard
Physical            Cleric
Spell               Mage
Trade               Monk
Weapon              Paladin
All                 Ranger
                    Thief
                    Warrior

DESCRIPTION:

This command shows you the abilities you have learned, how well you know the
abilities, and how much it will cost to improve them.  This command also
allows you to view the abilities available to all classes, including
prerequisite skills, practice point costs, and the levels at which classes can
learn abilities.

Specifying the "-l" flag sorts the ability list by the minimum experience
level required for a character of the specified class to first practice the
skill.  Otherwise, the ability list is sorted alphabetically.

Some abilities have prerequisites; skills you need to know before you can
acquire the new skill.  For example, before you can learn second attack you
must learn kick and punch.  The number next to the prerequiste is the level at
which you must have the prerequiste before getting the next skill.
