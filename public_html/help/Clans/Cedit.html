Help Topic: Cedit

SYNTAX

cedit                       : view existing titles, flags, holders.
cedit new <title>           : creates a new title
cedit delete #              : deletes an existing title
cedit # <+flag>             : sets a permission flag for that position
cedit # <-flag>             : removes a permission flag for that position
cedit # holder <name>       : sets the holder of a title
cedit # title <new title>   : changes the name of an existing title

FLAGS

Flags are case-sensitive.

F : May set flags on other title holders
C : May edit the clan's charter
N : may edit the clan's noteboard
R : may recruit members

DESCRIPTION

The cedit command allows you to create, manipulate, and delete clan leader
titles.

The cedit command may only be used by immortals who have the "clans"
permission flag, or by clan title holders who have the "F" permission flag. 
Mortals may not use the "new", "delete", or "title" options in any event.
