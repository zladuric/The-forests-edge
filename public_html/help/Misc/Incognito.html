Help Topic: Incognito

SYNTAX

option incognito # : sets your incognito level to a value between 0-3.

DESCRIPTION

Your incognito level controls which players are able to see your race, class,
level, sex, and religion using the 'whois' command.  It also controls who can
see your level and class in the 'who' command listing.

opt incog 0 : Your information is visible to everyone.
opt incog 1 : You're only visible to clan members and players you've
              befriended.
opt incog 2 : You're only visible to clan members.
opt incog 3 : You are totally incognito.

See also: befriend.
