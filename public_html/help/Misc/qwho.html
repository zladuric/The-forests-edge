Help Topic: qwho

SYNTAX

qwho : displays a shorter list of all players on

FLAGS

i : only lists those you recognize
b : only lists those you have befriended

DESCRIPTION

This command displays a short list of players.  For those using ANSI color,
players you have been introduced to are in yellow and those you have
befriended are in green.  For those curious, qwho stands for quick who.

See also: who
