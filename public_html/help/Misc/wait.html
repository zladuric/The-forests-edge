Help Topic: wait

SYNTAX

wait <seconds>    : insert a pause between queued commands

DESCRIPTION

The wait command allows you to insert a pause of up to 30 seconds between
queued commands.  Non-queued commands will bypass the wait, and disrupting
commands (such as "flee") will cancel the wait and everything else in the
command queue.

Note that, due to network buffering, the delay may appear to be up to 2
seconds longer than specified.  However, the command is actually executed at
the correct time, whether you see it immediately or not.

See also: queue.
