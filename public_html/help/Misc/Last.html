Help Topic: Last

SYNTAX

last immortal       : lists all immortals
last avatar         : lists all avatars
last <name>         : lists all players matching name
last clan [<name>]  : lists all member of abbreviated clan name
                      (own clan if <name> omitted)

DESCRIPTION

This command returns the real-world date and time of the player's (or
players') last logout from TFE.  Use the "time -z" command to adjust the
display to your timezone.

See also: clans, time.
