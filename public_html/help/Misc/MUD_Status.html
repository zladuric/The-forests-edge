Help Topic: MUD Status

This help file lists the status of all known features of the MUD that are not
fully operational.  This is not a bug list.

A red asterisk (*) indicates an item that is complete in code, but needs
builder support to finish.

General:
  Religion - various forms of player differentiation per diety being
    considered.
  PKill - expanded pkill rules being considered.
  Climate, Seasons, Weather - being implemented.
  Piety - being worked on.
  Fame - being worked on.
  Reputations - mostly implemented but not enabled.
  Poison - thief ability to coat weapons under consideration.
    Varying poison virulence under consideration.
  Vehicles - being considered.

Physical Skills:
  Heist - being implemented.
  Butcher - enabled, but needs better functionality.
  Disguise - being implemented.
  Circle - under consideration.
  *Counter Attack - replaced by riposte, which is complete, but needs a
    trainer.
  Hunt - being implemented.
  Gather - an alternative to hunt, for Ents - being considered.

Weapon Skills:
  No planned changes.

Spell Skills:
  *Conjure Elemental - implemented, summoned mobs need review.
    Needs casting sequence, reagents, exp levels, trainers.
  *Find Mount - good aligned mounts need to be upgraded/redone.
  *Request Ally - good and evil pets need to be created.
  Poison Cloud - abuse issues need to be resolved.
  Locust Swarm - abuse issues need to be resolved.
  Blinding Light - abuse issues need to be resolved.

Trade Skills:
  Leatherworking - not implemented, no plans yet.
  Metallurgy - not implemented, no plans yet.
  Woodworking - not implemented, no plans yet.

Player Classes:
  Druids - being implemented.

Races:
  Considering half-elves and half-orcs.
