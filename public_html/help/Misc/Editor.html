Help Topic: Editor

SYNTAX

<subj>                          : displays subject being edited
<subj> <text>                   : append text at bottom of whatever
                                  you are currently editing
<subj> <odd #>                  : deletes the specified line number
<subj> <odd #> <text>           : replaces the specified line with
                                  new text
<subj> <odd #>? <word>          : removes all occurences of word on
                                  the specified line
<subj> <odd #>? '<phr>'         : same as above, but allows spaces
<subj> <odd #>? "<phr1>"        : same as above, but allows single quotes
<subj> <odd #>? <word> <phr>    : replaces all occurences of word with
                                  phr on the specified line
<subj> <odd #>? '<phr1>' <phr2> : same as above, but allows spaces
<subj> <odd #>? "<phr1>" <phr2> : same as above, but allows single quotes
<subj> <even #>                 : inserts blank line here
<subj> <even #> <text>          : inserts text between lines here
<subj> #-#                      : formats lines in the specified range
<subj> ? <word>                 : searches and removes word
<subj> ? '<phr>'                : same as above, but allows spaces
<subj> ? <word> <phr>           : searches and replaces word with phr
<subj> ? '<phr1>' <phr2>        : searches and replaces phrase1 with
                                  phrase2 (be careful with search and
                                  replaces...editor will replace ALL
                                  cases it finds)
<subj> undo                     : copies the last edited text into the
                                  current buffer
<subj> #-# delete               : deletes all lines in the given range
<subj> indent                   : applies a C-like format to entire message
<subj> #-# indent               : applies a C-like format to lines in range
<subj> #-# cut                  : cuts lines in given range to paste buffer
<subj> #-# copy                 : copies lines in given range to paste
                                  buffer
<subj> #+                       : pastes contents of paste buffer at line
                                  number
<subj> spell                    : performs a basic spelling check
  
DESCRIPTION

<subj> is any one of the following: note, mail, desc, or journal.  There are
some editor functions that only work for a specific subject, so make sure to
read the help on note, mail, desc, and journal before going much further, if
you haven't already.

It is often simplest to write your message one sentence at a time.  This is so
that you can insert lines of text between what you have already written (use
even line numbers for this) and format it afterwards, using <subj> #-#, which
will make your sentences into organized paragraphs.

If you want to insert a line of text which happens to be the same as one of
the editor commands (e.g.  "note spell"), use a line number ("note 5 spell").

ADVANCED USAGE

Each player has two text buffers--the undo buffer and the paste buffer--which
can be used to make text editing easier.

The undo buffer holds the last state of the message prior to the last editor
command, even if it was just "<subj>" to list the message.  In fact "<subj>
undo" simply copies this buffer into the current message.  This allows copying
of an entire message from one place to another.  For example, if you have a
mail message you want to copy into a new note:

  mail edit #
  mail
  note new <subject>
  note undo

The paste buffer is used by the cut, copy, and paste (+) commands.  Cut and
copy place portions of the current message into the paste buffer.  Paste
("<subj> #+") inserts the contents of the paste buffer at the line number
indicated.  The paste buffer is not cleared when a new message is selected, so
it can be used to move or copy sections from one message to another.

EXAMPLES

> desc 1 This is a test

This adds "This is a test" as the first line of your description.  The line
number isn't really necessary if you are adding the first line, or appending
lines to your description.

> journal The mage in Chiiron asked me to bring him a cookie.

This simply appends this information to your journal.
