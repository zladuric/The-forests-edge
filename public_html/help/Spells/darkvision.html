Help Topic: darkvision

Some sages speculate that this spell functions by augmenting any light, no
matter how faint, while others speculate that it functions in the same way
that a bat's "vision" does, but is simply translated into a medium that the
caster can easily understand.  Regardless of how the spell operates, by using
this spell, the caster's eyes are magically enhanced so that the caster can
see in complete darkness, regardless of whether an object is radiating heat or
happens to be reflecting visible light.  Note that this spell provides no
additional ability to the caster to detect hidden, camouflaged, or invisible
creatures.
