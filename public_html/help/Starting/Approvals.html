Help Topic: Approvals

To be approved by an Immortal or an Avatar, you need to complete all
of the following: 

An appearance - see help Appearance.
A list of keywords - see help Keywords.
A description - see help Description.

(The identity command, with its verification fields, might prove helpful
in crafting your appearance).

All three need to be done at the same time, and before level 5.  You are
responsible for reading all these help files and following their
directions before asking an Immortal or Avatar to approve you.
Approvals are very time consuming and we ask for your cooperation.

When you are ready for approval, try request ava approval for
requesting an avatar's assistance in helping you get approved.  You can
type "who" and see AV next to the Avatars currently online.  If no
Avatars are online, use request imm <approval> for an immortal's
assistance.
   
Finally, your appearance and description should save, so take your time,
read the help files that govern how appearances and descriptions should
look, and come up with one that knocks people's woolen socks off.
