Help Topic: Appraise

SYNTAX

appraise <item>  : assesses the value of an item

DESCRIPTION

This skill allows you to assess the value of an item.  The higher your skill
and level, the more expensive items you can appraise.

This skill only determines the physical value of an item and is not a
substitute for the identify spell.  Thus it may determine the value of a gem
but it can not tell you what a magic scroll, wand , or phial does (etc).
This will allow appraisers to do such things as determine whether a gem is a
fake or not and once determined it will label the gem accordingly (as if an
identify spell had been cast on it).
