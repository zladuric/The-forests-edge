Help Topic: Buy

SYNTAX

buy <object>     : purchases that object
buy x*<object>   : purchases x number of that object

DESCRIPTION

   If you have enough money, and you are or are above the minimum level,
then you will buy the object from the list.  If you buy x*object, you
will buy x many of that object only if there are that many in stock. 
(also see syntax) Buying several of the same object before the shop
has time to restock will cause an increase in that object's price, and
some objects are not useable by certain classes/races.  With the
haggle skill, items can be bought for significantly less money.
