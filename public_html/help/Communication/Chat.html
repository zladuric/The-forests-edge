Help Topic: Chat

SYNTAX

chat <message> : send <message> to the mud general chatter channel
chat off       : turns the chat channel off
chat on        : turns the chat channel on
chat           : display last 5 chats

DESCRIPTION

   Use this channel for idle talk about anything you want, but please
stay in character, of course.  If trying to get the attention of an
immortal use gossip.  Use of this channel requires gossip points; it
costs one gossip point per use of chat.  These points accumulate over
time, and non-use of any channel.  To see how many points your character
has accumulated, type "score"; the points are listed under Gsp_points.

Only "allied" race players will be able to hear your chat, and you will not
hear the chat of non-allied players.  The light/dark system is currently used
for this division.

See also: channels, score
