Help Topic: Gossip

SYNTAX

gossip <message> : send <message> to the global gossip chat channel
gossip off       : turns the gossip channel off
gossip on        : turns the gossip channel on

DESCRIPTION

  Use this channel for talking about mud related topics "in character". 
Use of this channel also requires gossip points.  It costs five gossip
points to gossip once.  The points accumulate over time, and unuse of the
gossip channel.  To see how many gossip points your character has
accumulated, type "score" (see help score); the points are listed
under Gsp_points . 

See also: help channels, help gsp_points
