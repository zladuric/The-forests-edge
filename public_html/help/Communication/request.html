Help Topic: request

SYNTAX

request              : lists pending requests
request imm <reason> : requests an imm
request ava [reason] : requests an avatar
request cancel       : cancel all requests by you

DESCRIPTION

This command is for mortals to request the attention of an avatar or
immortal.  Please make your reasons as precise as possible.  In simple
cases hopefully removing any need for conversation.


When an immortal has helped you please type request cancel to clear it.

When using this command, asking the question in the request is a good
idea.

i.e.  request imm the goblin spearman is blinding himself

As opposed to: request imm problem with mob
